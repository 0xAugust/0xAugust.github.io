    • 基本设置

    git config --list

    配置用户信息

    git config --global user.name "XXX"
git config --global user.name "XXXXX@gmail.com"

    版本初始化

    git init # 初始化本地新仓库，并且会在工作目录下生成.git的隐藏文件夹

    将文件添加至缓存

    git add . # '.'为将所有文件添加至缓存中

    为修改过的文件添加说明内容

    git commit -m "这里是说明"

    查看历史提交记录

    git log

    回滚至某个commit的Sha码

    git reset --hard HEAD^       # 退至上个版本
git reset --hard HEAD~n      # 退至n次提交之前
git reset --hard commit_id   # 推至指定commit的Sha码

    远程仓库

    创建SSH KEY

    ssh-keygen -t rsa -C "xxxxx@gmail.com" # Windows系统使用git bash运行此命令

    创建后路径为: C:\Users\自己的用户名.ssh目录

    其中id_rsa为私钥 id_rsa.pub为公钥

    登录Github

    登录之后点击头像

    Settings > SSH keys >New SSH key

    将公钥的内容复制至github中。

    创建项目

    在Github中创建一个仓库，将仓库地址复制，添加远程仓库

    git remote add origin https://github.com/Knsec/Knsec.github.io.git

    推送至远程仓库

    git push -u origin master
