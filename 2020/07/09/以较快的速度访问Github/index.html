    修改Host文件

    C:\Windows\System32\drivers\etc\host

    如果是第一次修改则需要去掉只读属性，并且更改文件的高级安全选项，使得自己的用户有权限修改host文件

    修改过后保存并把刚刚修改的属性改回只读即可

    有些人会觉得修改host文件是一个安全性较差的方法

    其实修改host文件本质上就是省去了dns解析的步骤，但是当我们把Google修改到第三方ip地址时，这个做法确实是不能保证数据传输过程中的安全性。

    至此，可以加快访问Github的速度，并且顺带解决头像、图片加载不出来的问题

    gitee

    许多人比较推荐国内的这个类似gitee，但因个人原因，不太愿意使用这个方法。
